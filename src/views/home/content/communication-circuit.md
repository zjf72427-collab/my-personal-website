## Ch.01 高频小信号放大器

### 谐振放大器基本原理

高频小信号放大器以 **LC 并联谐振回路** 作为负载，利用谐振回路的选频特性实现对特定频率信号的放大。

谐振频率：
$$f_0 = \frac{1}{2\pi\sqrt{LC}}$$

品质因数 Q 决定回路选择性：
$$Q = \frac{f_0}{BW} = \frac{R_p}{\omega_0 L}$$

Q 越高，通频带 BW 越窄，选择性越好，但增益带宽积不变。

### 单调谐 vs 双调谐

| 指标 | 单调谐 | 双调谐 |
|------|--------|--------|
| 矩形系数 | 差（~9.95） | 好（~3.16） |
| 实现难度 | 简单 | 需调谐两个回路 |
| 适用场景 | 一般选频 | 高性能接收机 |

### 放大器稳定性分析

高频放大器存在内部反馈，可能导致自激振荡。稳定化措施：

- **单向化**：通过中和电容抵消 BJT 内部 Cbc 的反馈
- **失配法**：降低源阻抗或负载阻抗，减小反馈影响
- **共基接法**：Cbc 不跨接输入输出，稳定性优于共射

### 噪声系数 NF

$$NF = \frac{S_{in}/N_{in}}{S_{out}/N_{out}} = \frac{SNR_{in}}{SNR_{out}}$$

多级放大器级联时，噪声系数由第一级主导（Friis 公式）：
$$NF_{total} = NF_1 + \frac{NF_2 - 1}{G_1} + \frac{NF_3 - 1}{G_1 G_2} + \cdots$$

因此 **第一级放大器的低噪声设计** 至关重要。

## Ch.02 LC 振荡器与频率稳定性

### 振荡的巴克豪森准则

产生自激振荡的必要条件：

1. **相位条件**：环路总相移 = 0°（或 360° 整数倍）
2. **振幅条件**：环路增益 |AF| ≥ 1（起振时 > 1，稳定时 = 1）

### 三点式振荡器判断法则

「射同基反」规则：与发射极（Source）相连的两个电抗元件**同性质**，与基极（Gate）相连的电抗元件**异性质**。

```
电容三点式（Colpitts）：
  C1（E-B 间）, C2（E-地 间）, L（B-地 间）
  特点：频率稳定性好，高频性能佳

电感三点式（Hartley）：
  L1（E-B 间）, L2（E-地 间）, C（B-地 间）
  特点：输出幅度大，但谐波丰富
```

### 克拉泼振荡器（Clapp）

在科皮兹基础上串入 C3（C3 << C1, C2）：

$$f_0 \approx \frac{1}{2\pi\sqrt{LC_3}}$$

振荡频率主要由 C3 决定，C1、C2 的寄生变化影响小，**频率稳定度显著提高**。

### 晶体振荡器

石英晶体的等效电路中存在两个谐振点：
- **串联谐振 fs**：等效阻抗最小（近似短路）
- **并联谐振 fp**：等效阻抗最大（近似开路）

fs 与 fp 极为接近（差值 < 1%），晶体在此频段内呈感性，可替代三点式振荡器中的电感，实现 10^-6 量级的频率稳定度。

## Ch.03 调幅 AM 与包络检波

### 普通调幅（AM）

$$s_{AM}(t) = [A_0 + m_a \cdot m(t)] \cos(\omega_c t)$$

调幅系数 $m_a = \Delta A / A_0$，要求 $m_a \leq 1$（过调制导致失真）。

AM 频谱包含：载波 + 上边带（USB）+ 下边带（LSB），带宽 BW = 2fm。

**功率效率低**：有用信息功率（边带）占总功率比例最高仅 33.3%（ma=1 时）。

### DSB 与 SSB

- **DSB**（双边带抑制载波）：去掉载波，效率 100%，但需相干解调
- **SSB**（单边带）：只保留一个边带，带宽减半（BW = fm），频谱利用率最高

### 包络检波电路

```
输入 AM 信号 → 二极管整流 → RC 低通滤波 → 输出包络

关键参数约束：
  1/fc << RC << 1/fm
  （RC 太小：纹波大；RC 太大：惰性失真）

惰性失真条件（RC 过大导致跟不上包络下降）：
  RC ≤ √(1 - ma²) / (ma · ωm)
```

## Ch.04 角度调制 FM/PM

### FM 与 PM 的区分

| | 调频 FM | 调相 PM |
|--|---------|--------|
| 瞬时频偏 | 正比于 m(t) | 正比于 dm/dt |
| 最大频偏 | Δf = kf · Am | Δφ = kp · Am |
| 调制指数 | mf = Δf/fm | mp = kp · Am |

### Carson 带宽公式

$$BW_{FM} = 2(\Delta f + f_m) = 2 f_m (m_f + 1)$$

- mf << 1（窄带 FM）：BW ≈ 2fm（类似 AM）
- mf >> 1（宽带 FM）：BW ≈ 2Δf

### 鉴频方法

```
斜率鉴频：利用谐振曲线斜率将频率变化转为幅度变化
  优点：简单  缺点：线性范围小

相位鉴频（Foster-Seeley）：
  利用 FM 信号通过移相网络后的相位变化实现鉴频

比例鉴频：在相位鉴频基础上增加稳幅电容，
  自动抑制寄生调幅（AM 噪声），无需限幅器
```

### 预加重与去加重

FM 系统中高频噪声功率谱密度高于低频，导致高频 SNR 差。

- **预加重**（发送端）：对调制信号高频分量提升约 +6dB（RC 高通）
- **去加重**（接收端）：对应低通滤波还原平坦频谱
- 效果：高频 SNR 改善约 **13dB**（广播 FM 标准：τ = 75μs）
