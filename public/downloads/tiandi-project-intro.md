# 天地协同管理平台项目介绍

## 项目概述

天地协同管理平台是一个面向低空设备与地面感知设备的可视化管理系统，围绕“设备接入、实时监测、智能预警、数据分析、巡检报告”构建完整的业务闭环。平台可用于无人机、传感器、摄像头、充电站等设备的统一管理，支持在地图与图表大屏中实时展示设备状态、轨迹、告警和环境数据。

该项目同时具备演示、教学和原型验证价值，既能作为“天地协同”场景的展示平台，也能作为 SC171 开发板、传感器设备和模拟器联调的实验环境。

## 核心能力

- 实时设备管理：支持设备注册、分组、状态展示和在线监测。
- 地图可视化：在地图上动态显示设备位置、轨迹与覆盖范围。
- 告警中心：支持超速、低电量、烟雾、围栏越界、离线等多类告警。
- 智能分析：支持 AI 诊断、异常分析、趋势预测和巡检报告生成。
- 数据图表：内置设备类型统计、速度曲线、温度曲线、今日告警统计等可视化组件。
- 数据导出：支持告警与遥测数据导出为 CSV。
- 权限与登录：支持登录页、用户认证和密码修改。
- 真实设备对接：可通过 HTTP API 接入 SC171 开发板或其他终端设备。

## 文件夹说明

### `.claude/`

该目录用于本地 AI 协作与开发辅助配置。

- `settings.local.json`：本地输出风格配置，当前以“解释清晰、便于理解”为主。

这个目录不承载业务功能，但体现了项目开发过程重视文档表达、内容整理和协作效率。

### `code/`

该目录是项目核心代码区，包含后端服务、前端页面、模拟器、部署文件和数据库文件。

- `main.py`：平台后端主程序，提供设备管理、遥测接收、告警处理、围栏管理、统计分析、报表生成、用户登录等 API。
- `simulator.py`：设备数据模拟器，用于模拟无人机或传感器向平台持续上报数据。
- `guide.py`：学习指南与项目讲解脚本，适合教学展示或快速理解系统结构。
- `make_ppt.py`：与汇报材料生成相关的脚本。
- `README.md`：项目说明文档，包含启动方式、API 使用方法和技术栈说明。
- `requirements.txt`：Python 依赖清单。
- `Dockerfile`：镜像构建配置。
- `docker-compose.yml`：容器编排配置。
- `tiandi.db`：本地 SQLite 数据库，保存设备、遥测、告警、围栏、用户、日报等数据。

#### `code/static/`

前端静态资源目录，构成平台网页界面。

- `index.html`：主界面，包含地图、设备列表、告警列表、AI 面板、图表面板、告警历史和规则配置等模块。
- `login.html`：登录页面。
- `css/style.css`：平台样式文件。
- `js/app.js`：前端核心逻辑，负责地图渲染、图表更新、WebSocket 实时通信、设备管理、告警处理、围栏交互、轨迹回放、AI 分析和报表展示。

#### `code/code/`

该子目录主要保存补充脚本与汇报材料。

- `make_ppt.py`：补充版汇报生成脚本。
- `天地协同管理平台_汇报.pptx`：项目汇报演示文件。

#### `code/__pycache__/`

Python 运行时自动生成的缓存目录，不属于核心业务内容。

## 项目特色

1. 将“天端设备”和“地端设备”统一纳入一个可视化平台。
2. 同时具备实时监测、智能预警、地图展示和数据报表能力。
3. 支持模拟器联调，便于教学、答辩和快速演示。
4. 支持围栏、轨迹、AI 分析、告警规则配置等扩展功能。
5. 兼具代码系统、汇报材料和研究文档，适合完整项目展示。

## 简短对外介绍文案

天地协同管理平台是一套面向无人机、传感器、摄像头等设备的实时监测与智能预警系统。平台基于 FastAPI、WebSocket、Leaflet 和 ECharts 构建，支持设备接入、地图展示、告警处理、轨迹回放、围栏管理、AI 巡检分析和数据导出。项目同时配套了调研文档、实施方案和汇报材料，适合用于科技创新展示、教学实践和原型验证。
